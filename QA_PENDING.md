### рҹӣ пёҸ New QA/admin commands (this build)
- **`/gp augment set <function> <level>`** вҖ” the no-UI install path: hold a **Kernel** (breeding upgrade), run it to set an augment, then slot the Kernel into a pasture. Selectors use the level as an index вҖ” e.g. `/gp augment set nature 4` вҶ’ Adamant, `/gp augment set ball 4` вҶ’ master_ball; binaries `set ability 1` / `set egg_move 1`. `/gp augment list` shows the held Kernel; `set вҖҰ 0` removes one; `/gp augment clear` strips all. **This is how you install Q31вҖ“Q34.**
- **`/gp goal set <species|any> <shiny> <minPerfectIvs> <count>`** вҖ” the breeding-goal tracker (Q37); `/gp goal` shows progress, `/gp goal clear` clears.
